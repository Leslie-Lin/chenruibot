# chenruibot
陈睿机器人，有简单的秘书功能，会发色图，基于IOTBOT开发

## 使用方法：

启动iotbqq，按照它的使用方法登录qq号，参考https://github.com/IOTQQ/IOTQQ

在该程序中的config.json输入大号qq（不用秘书功能可以不输），小号qq，iotqq的api地址，运行myqqbot.py即可

## 文件说明
myqqbot.py:主程序

text.py:处理接受的信息并调用相关函数做出反应

method文件夹:所有的机器人功能
